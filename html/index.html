Opus File Converter (opusFC) Documentation¶

Introduction¶

The opusFC module converts binary OPUS files into NumPy arrays. It is primarily intended to be used with the orange-infrared add-on for the Orange3 Data Mining suite.

Usage¶

To determine the contents of a file:

f = /path/to/opusfile.0
dbs = opusFC.listContents(f)
print(dbs)

Load the first datablock in dbs:

data = opusFC.getOpusData(f, dbs[0])

Access loaded data and parameters:

data.x                    # X coordinates
data.parameters['SNM']    # Sample name
data.description          # Description of data object
data.paramDict['SNM']     # String description of SNM

Contributing¶

opusFC is distributed as a compiled python/cython module only. Feedback, bug reports, and other contributions are welcomed at the python-opusfc-dist github repo or by contacting the authors directly.

Reference¶

class opusFC.DataReturn¶

Data object returned by .getOpusData().

Attributes will depend on thee requested datablock and can be one of 2D, 3D or 3DTRC:

  • 2D - Single spectrum
  • 3D - Hyperspectral cube of image/map data
  • 3DTRC - Map(s) of a single spectral feature/integral/calculation
dataType¶

str – The datablock type

dimension¶

str – Datablock dimensions (2D or 3D)

derivative¶

str – Datablock derivative (NONE, 1DER, 2DER, NDER)

description¶

str – Description of the :class:DataReturn object attributes

parameters¶

dict – Dictionary of relevant data/sample/instrument parameters

x¶

ndarray – 1D numpy array of spectrum x coordinates

maxY¶

float – Maximum y value

minY¶

float – Minimum y value

y¶

ndarray, 2D – 1D numpy array of y values

mapX¶

ndarray, 3D – 1D numpy array of X map coordinates

mapY¶

ndarray, 3D – 1D numpy array of Y map coordinates

spectra¶

ndarray, 3D – 3D numpy array of spectral maps dataObject.spectra[row][column][x]

labels¶

list of str, 3DTRC – List of trace block names

traces¶

list of ndarray, 3DTRC – 3D numpy array of trace maps with same index as labels: dataObject.traces[:,:,index]

opusFC.getOpusData()¶

This function returns a DataReturn object containing the data and data parameters for the specified datablock in the given file.

Parameters:
  • path (path) – Path to file
  • datablock (tuple of str) – tuple(datablocktype, dimensions, derivative) as returned by .listContents()
Returns:

Object containing the requested data and associated parameters. Data structure is as specified in data.description and can be one of the three types described in DataReturn
Return type:

DataReturn
opusFC.help()¶

Print a help message containing the available functions.

opusFC.isOpusFile()¶

Tests if the given file is an OPUS data file.

Parameters:path (path) – Path to file
Returns:True if an OPUS file
Return type:bool
opusFC.listContents()¶

This function returns a list of datablocks for a given file. Each datablock is identified by blocktype, dimensions and derivative.

Parameters:path (path) – Path to file
Returns:list of tuple – List containing a (datablocktype, dimensions, derivative) tuple for each datablock in the file
Return type:str, str, str
